# Memory System

This `.memory/` folder contains project documentation and tracking information to maintain context across development sessions.

## Files

- **decisions.md** - Technical and architectural decisions with reasoning
- **analysis.md** - Codebase structure and analysis
- **doubts.md** - Open questions and points to clarify
- **progress.md** - Progress journal and milestones
- **cleanup-log.md** - Detailed log of what was removed and why
- **build-issues.md** - Build and startup issues encountered with solutions
- **roadmap.md** - Planned features and next steps
- **documentation.md** - API and application documentation

## Purpose

This memory system helps track decisions, maintain project context, and provides a reference for future development work.
